About

Westsmith is a software consultancy based in Winchester, UK. We build custom software, integrate AI and design cloud systems for businesses that need reliable, well-crafted solutions delivered without the overhead of a large agency.

Founded in 2019 by Daniel Ball, the company was initially set up to deliver contract work for IBM. Since then we've worked with government departments, healthcare providers and large enterprises including the Office for National Statistics and Vodafone.

We work with clients across the UK on a project or retainer basis. Whether that's a greenfield build, modernising something legacy or embedding AI into an existing product, the approach is the same. Close collaboration, clear communication and a focus on shipping something that works.

AI integration is an active and growing part of our work, with hands-on production experience applying Anthropic's Claude in real products.

Core skills

  • Backend development using Python and modern API frameworks like FastAPI, Flask and Django
  • Cloud infrastructure on GCP and AWS, including serverless architecture, container orchestration and infrastructure as code
  • AI integration using Anthropic's Claude, from RAG systems and document processing to LLM-powered report generation
  • Full-stack delivery from prototypes to production, with a strong grasp of front-end templating, CI/CD and testing pipelines
